# Barrel Tool

## [BarrelType](../Assets/Scripts/BarrelTypes.cs)
In this tool you can assign scriptableObject BarrelType to odject. Barreltype objects have custom inspector allowing to change properties - Color, Radius and Damage.

Selecting object in scene draws gizmo showing barrel explosion radius.

## [BarrelManager](../Assets/Scripts/BarrelManager.cs)
Gather all barrels. Have custom editor allowing to select and display specific barrel type. On selecting in scene draws guide lines to selected type barrels.
